[absolute rules](#absolute-rules) --- ## typography ### primary typeface **poppins** — google fonts — loaded via @import in chassis.css ```css @import url('https://fonts.googleapis.com/css2?family=Poppins:wght@100;200;300;400;500;600;700;800;900&display=swap'); font-family: 'Poppins', Arial, Helvetica, sans-serif; ``` fallback stack: `Arial, Helvetica, sans-serif`. all nine weights imported. ### weight usage | weight | name | usage | |--------|------|-------| | 100–200 | thin / extra-light | large display headlines, hero h2 | | 300 | light | body text, captions, descriptions — default reading weight | | 400 | regular | navigation links, labels, ui text | | 500 | medium | nav links, button text | | 600 | semi-bold | h1, h3, link lists, strong emphasis | | 700–900 | bold–black | reserved — sparingly for maximum contrast moments. people biog person name uses 700. | ### typographic scale — destination engine | element | size | weight | colour | |---------|------|--------|--------| | h1 (.text-rail) | 48px / 38px mobile | 600 | #0054dc | | h2 (.text-rail) | 28px | 200 | #191919 | | h3 (.text-rail) | 24px | 600 | #0054dc | | body (p) | 18px | 300 | #333333 | | blockquote | 32px / 24px mobile | 300 | #444444 | | caption | 15px | 300 | #666666 | | nav links | 13px | 500 | #ffffff | | footer links | 14px | 300 | #ffffffb3 | | .person-name (biog) | 32px | 700 | #191919 | ### absolute typography rules - **no serif typefaces anywhere in the framework.** poppins only. - the blockquote quotation marks use `font-family: Georgia, serif` for the decorative glyph only — this is the sole exception and must not be extended. - **no uppercase text** in ui, navigation or body copy. lowercase is a deliberate design decision. - hierarchy through weight, scale and space only. --- ## colour palette the palette is deliberately restrained. two brand colours, one dark ground, one light ground. all other values are neutrals. **strict hex-only — no rgb() or rgba() in production code. use hex-8 for transparency.** ### brand colours | name | hex | css variable | usage | |------|-----|-------------|-------| | brand blue | `#0054dc` | `--brand-blue` | primary brand, nav bg, headings, links | | brand orange | `#ff6600` | `--brand-orange` | hover states, orange rule, interactive feedback | | midnight blue | `#0a254a` | `--footer-bg` | footer, lightbox bg, projector bg | | stone | `#f5f5ef` | `--stone-bg` | page background, caption backgrounds | | charcoal | `#191919` | `--charcoal` | h2, body text baseline | ### supporting neutrals | hex | usage | |-----|-------| | `#ffffff` | paper sheet, primary white | | `#dddddd` | standard rule, image outlines, borders | | `#f4f3ef` / `#f9f8f4` | caption bg, resource card bg | | `#666666` | caption text, label text, biog contact meta label | | `#555555` | resource description text | | `#333333` | body text default | | `#888888` | blockquote attribution | | `#bbbbbb` | theatre environment text | | `#ffffff80` | footer address (hex-8: 50% opacity) | | `#ffffffb3` | footer nav links (hex-8: 70% opacity) | | `#ffffff26` | brand-blue-light (hex-8: 15% opacity) | ### known issue brand orange has two values across modules: `#ff8c00` in chassis, `#ff6600` in destination and theatre. destination takes precedence on content pages. resolve to `#ff6600` in next major version. ### saturation lock brand blue is fixed at `#0054dc` and enforced with `!important` on the nav element. do not modify. --- ## voice & tone ### the voice | quality | in practice | |---------|------------| | direct and considered | no waffle. say the thing. then stop. | | warmth without sentimentality | human, not corporate. engaged, not gushing. | | intellectual curiosity without jargon | the idea matters more than demonstrating the vocabulary. | | craft visible but worn lightly | the work speaks. you don't need to explain it. | | serious about the work, not precious | rigour without reverence. it's allowed to be funny. | | humour earned, not performed | wit is a marker of intelligence — but it must land naturally. | ### what to avoid - corporate language - breathless enthusiasm about technology for its own sake - over-explaining what the work is trying to do - nostalgia as a destination (the past is load-bearing, not load-limiting) - consolation dressed as strategy ### the four registers | register | cadence | character | |----------|---------|-----------| | essay | quarterly | long form. memoir, influence, cultural excavation. cannot be forced. | | arts essay | monthly (alternating) | houdini window production diary. honest about the struggle. | | connections | monthly (alternating) | associative. gallery-minded. following the thread without announcing the destination. | | dispatch | weekly | 200–300 words. low threshold. the pulse. | --- ##
